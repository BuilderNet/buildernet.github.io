## TEE VM image downloads

TEE virtual machine images can be downloaded from **[builder-artifacts.flashbots.net](https://builder-artifacts.flashbots.net)**

<hr class="clean" />

## Allowed TEE measurements

Nodes are required to run a specific boot chain, codebase and configuration, verified using [TEE attestation and measurement mechanisms](verifiable-system-integrity).

:::info

Allowed TEE measurements are published at **[measurements.builder.flashbots.net](https://measurements.builder.flashbots.net)**

:::
