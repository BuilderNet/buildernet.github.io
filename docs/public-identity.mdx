### Key identifiers

Blocks built by BuilderNet nodes expose these key identifiers:

| Identifier            | Description                                                                                                                                                                                                                                                                                                  |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Coinbase signing key  | Used to sign the block and for the proposer payment transaction. Also to accrue MEV value, which is later redistributed to users. Initially the coinbase signing key is common across builder instances (which might change in the future) and has the address `0xdadB0d80178819F2319190D340ce9A924f783711`. |
| Relay submission keys | Two BLS keys are used by each builder instance to sign bids submitted to relays, one for optimistic mode and one for non-optimistic submissions. These keys are unique per builder node.                                                                                                                     |
| Extra data            | `BuilderNet (<Operator>)`                                                                                                                                                                                                                                                                                    |
